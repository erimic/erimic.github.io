Hexo常用命令

init

$ hexo init [folder]

新建一个网站。如果没有设置 folder ，Hexo 默认在目前的文件夹建立网站。

new

$ hexo new [layout] <title>

新建一篇文章。如果没有设置 layout 的话，默认使用 _config.yml中的 default_layout 参数代替。如果标题包含空格的话，请使用引号括起来。

NexT如何在主页中使用显示更多按钮

发表于 2017-03-15 | | 阅读次数

如何设置[阅读全文]按钮

默认情况下Hext在首页中显示文章的全部内容，如果需要设置一篇文章的部分内容，并提供链接跳转到全部内容页面，NexT提供了三种方式来控制文章在首页的显示。

  1. 在文章中使用<!-- more -->手动截断(推荐方式)

  2. 在文章的front-matter中添加description,并提供文章摘录

  3. 自动形成摘要，在thems/next/_config.yml中添加

    auto_excerpt:
  enable: true
  length: 150

    默认截取的长度为 150 字符，可以根据需要自行设定

建议使用 <!-- more -->（即第一种方式），除了可以精确控制需要显示的摘录内容以外， 这种方式也可以让 Hexo 中的插件更好的识别。
