11gR2 for Standalone Server Installation

发表于 2017-03-16 | | 阅读次数

本文主要记录Oracle Database 11gR2单机安装。

准备

安装介质下载

Installation Type Zip File
OracleDatabase (includes Oracle Database and Oracle RAC) ==Note: you mustdownload both zip files to install Oracle Database.== p13390677_112040_platform_1of7.zip p13390677_112040_platform_2of7.zip
Oracle GridInfrastructure (includes Oracle ASM, Oracle Clusterware, and Oracle Restart) p13390677_112040_platform_3of7.zip
Oracle Database Client p13390677_112040_platform_4of7.zip
Oracle Gateways p13390677_112040_platform_5of7.zip
Oracle Examples p13390677_112040_platform_6of7.zip
Deinstall p13390677_112040_platform_7of7.zip

安装

安装GRID

上传p13390677_112040_platform_3of7.zip到服务器

SSH ORACLE USER

cd /u01/app/ins
unzip –o p13390677_112040_Linux-x86-64_3of7.zip

GNOME ORACLE USER

cd /u01/app/ins/grid
./runInstaller

使用root用户登录执行

sh /u01/app/oraInventory/orainstRoot.sh
sh /u01/app/oracle/product/11.2.0/grid/root.sh

配置FRA磁盘组

cd /u01/app/oracle/product/11.2.0/grid/bin
./asmca

安装数据库

将p13390677_112040_Linux-x86-64_1of7.zip、p13390677_112040_Linux-x86-64_2of7.zip上传到服务器

cd /u01/app/ins
unzip –o p13390677_112040_Linux-x86-64_1of7.zip
unzip –o p13390677_112040_Linux-x86-64_2of7.zip
cd /u01/app/ins/database
./runInstaller

使用root用户登录执行

sh /u01/app/oracle/product/11.2.0/dbhome_1/root.sh

使用DBCA创建Oralce DB

cd /u01/app/oracle/product/11.2.0/dbhome_1/bin
./dbca
