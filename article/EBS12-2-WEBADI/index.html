发表于 2017-07-06 | | 阅读次数

Microsoft Office Integration with Oracle E-Business Suite 11i and R12 NoteId:1077728.1

OFFICE 2016设置

  1. 点击 文件>选项
  2. 点击 信任中心>信用中心设置
  3. 在宏设置中，选择“禁用所有宏，并发出通知”
  4. 在宏设置中，选择“信任对VBA工程对象模型的访问”
  5. 点击 确定

添加桌面集成管理器职责

配置集成器

  • [ ] 桌面集成管理器 > 管理集成器

    ###### ![1499333224603](/images/blog/1499333224603.png)

  • [ ] 创建集成器

    | 属性    | 值              |
| ----- | -------------- |
| 集成器名称 | 管理人才库数据导入      |
| 内部名称  | CUX_PA_HS_M    |
| 应用    | cux custom app |

> 在“创建文档”页中显示（是否在“桌面集成”职责“创建文档”功能中供选择）
>
> 不选安全性规则（用于选择该集成器对应的功能，可先空着，后面再补）

![1499333756272](/images/blog/1499333756272.png)

  • [ ] 创建接口

    | 属性      | 值                     |
| ------- | --------------------- |
| 接口名称    | Upload Manager Source |
| 接口类型    | API-过程                |
| 程序包名称   | cux_pa_hrsource_pkg   |
| 过程/函数名称 | import_manager_source |
| API 返回  | FND 消息代码              |

![1499333892723](/images/blog/1499333892723.png)

> 选择“FND消息代码”接口程序中通过以下代码返回错误
>
> ```
> fnd_message.set_name('PER','AC_WEBADI_MESSAGE'); 
> fnd_message.set_token('MESSAGESTR','生效日期不能为空'); --改成要提示的具体消息       
> fnd_message.raise_error; 
> ```
>
> 其中'AC_WEBADI_MESSAGE'是自行定义的一个消息，为了显示需要提示的具体消息。

​

    | 属性   | 值                                 |
| ---- | --------------------------------- |
| 内容名称 | Project Manager Source            |
| 内容类型 | SQL 查询                            |
| 查询   | select * from cux_pa_hrsource_m_v |

![1499338503952](/images/blog/1499338503952.png)

| 加载程序 | 自模板  |

![1499338778555](/images/blog/1499338778555.png)

点击“创建”

创建组件

  • [ ] 桌面集成管理器 > 管理组件

    | 属性   | 值                  |
| ---- | ------------------ |
| 显示名  | 项目管理人才库组件          |
| 内部名称 | PROJ_MANAGE_SOURCE |
| 应用   | cux custom app     |
| 组件类型 | 表值列表               |
| 组件实体 | BneOAValueSet      |

![1499347998488](/images/blog/1499347998488.png)

​

> 内部名称需小于20个字符

  • [ ] 设置组件参数列表

    | 参数                  | 参数值                  | 参数说明                       |
| ------------------- | -------------------- | -------------------------- |
| table-column-alias  | emp_num              |                            |
| table-column-sort   | ASCENDING            | 排序，使用默认值                   |
| table-columns       | emp_number,last_name | 按次序列出需要显示的字段名，以西文逗号隔开      |
| table-headers       | 员工编号,姓名              | 按次序列出需要显示的字段名对应的名称，以西文逗号隔开 |
| table-select-column | p_segment1           | 返回到的接口字段名                  |
| window-caption      | 选择人员                 | 弹出窗口的标题                    |
| window-height       | 500                  | 调整弹出窗口的高度                  |
| window-width        | 400                  | 调整弹出窗口的宽度                  |

定义功能项

系统管理员>应用产品>功能

属性 值
功能 CUX_PA_HS_M
用户功能名 项目管理人才库数据导入
类型 SSWA servlet函数
参数 bne:page=BneCreateDoc&bne:viewer=BNE:EXCEL%25&bne:reporting=N&bne:integrator=CUX:CUX_PA_HS_M_XINTG

需要找出Integrator Code:CUX:CUX_PA_HS_M_XINTG
